# 登录接口

## 1. 登录请求

**接口地址：** `POST /api/auth/login`

**请求体：**
```json
{
    "account": "zhangsan",
    "password": "123456"
}
```

## 2. 登录成功响应

**响应体：**
```json
{
    "code": 200,
    "message": "登录成功",
    "data": {
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "userId": 1,
        "username": "张三",
        "email": "zhangsan@example.com"
    }
}
```

## 3. 登录失败响应

**响应体：**
```json
{
    "code": 401,
    "message": "账号或密码错误",
    "data": null
}
```
